Sistema de Biblioteca:

Este é um sistema de gerenciamento de biblioteca desenvolvido em Python usando Tkinter para a interface gráfica e SQLite para o banco de dados.

Funcionalidades Principais:
Login de Administrador: O sistema requer autenticação de administrador para acessar suas funcionalidades.
Cadastro de Livros: Permite adicionar novos livros ao acervo da biblioteca, incluindo informações como nome do livro, autor, gênero, localização na estante e quantidade disponível.
Cadastro de Alunos: Permite adicionar novos alunos ao sistema, registrando informações como nome, série, turma, sala e endereço.
Empréstimo de Livros: Possibilita o empréstimo de livros para alunos cadastrados, registrando os detalhes do empréstimo, como o aluno, livro emprestado, data de empréstimo e data de devolução prevista.
